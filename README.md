# MLCS

A Crate for finding the MLCS of an arbitrarily big number of strings

## Testing

1. **Clone the repository:**

    ```sh
    git clone https://github.com/epita-rs/MLCS.git
    cd MLCS
    ```

2. **Start the tests**

    ```sh
    cargo test --release
    ```
3. **Start the performance benchmark**

    ```sh
    cargo bench
    ```
